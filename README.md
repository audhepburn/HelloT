# HelloT

A world clock and stopwatch iOS app built with SwiftUI.

## Features

- **World Clock** — View analog clocks for Beijing, New York, Paris, and Tokyo simultaneously. Tap any clock to expand it into a full-screen detail view with a large dial, digital time, date, and timezone info.
- **Stopwatch** — Start, stop, and lap timer with history (up to 10 records). Swipe to delete individual records or clear all at once.
- **System Settings** — Toggle dark/light mode and flat/classic UI style.
- **Flat UI Mode** — Switch from analog clock grid to a clean list-based layout with color-coded rows.
- **Persistent Settings** — Theme, UI style, and stopwatch history are saved across app launches.

## Requirements

- Xcode 15+
- iOS 17+
